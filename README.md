# Multi-Channel Automation SaaS

A Next.js SaaS MVP for company-scoped contacts, groups, manual posting, scheduled automations, and delivery logs.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase PostgreSQL
- Gmail SMTP for email
- Twilio for SMS
